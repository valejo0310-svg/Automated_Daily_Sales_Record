# Sales Record System – RIWI Store

## Project Description
This project consists of a simple sales recording system. The program allows the user to register product sales, store the information, and generate a final summary of the registered transactions.

The main objective of this project is to apply fundamental programming concepts such as modular programming, control structures, and data organization.

## Programming Language
The system was developed using the following language:

- **Python**

## Project Structure
The project is organized using a modular structure to improve readability and maintainability.

For this project it was need to use: functions, while and for loops, list and dictionarys that would allow the program to run efficiently.
